    Module 3 : Pratique TensorFlow/Keras pour BTS SIO

    Introduction

    Ce module est entièrement axé sur la pratique et conçu spécifiquement pour des étudiants de BTS SIO. Vous allez apprendre à utiliser TensorFlow/Keras à travers des exercices concrets, avec deux options disponibles : - Version Google Colab (aucune installation requise) - Version locale (pour ceux qui souhaitent installer les outils sur leur machine)

    Prérequis

    Pour la version Google Colab

    • Un compte Google
    • Navigateur web récent
    • Aucune installation requise

    Pour la version locale

    • Python 3.7 ou supérieur installé
    • Pip (gestionnaire de paquets Python)
    • Éditeur de code (VS Code recommandé)

    Partie 1 : Premier modèle avec TensorFlow/Keras (1h30)

    Objectifs

    • Comprendre les bases de TensorFlow/Keras
    • Créer et entraîner un modèle simple
    • Visualiser les résultats

    Version Google Colab

    1. Accès au notebook
    2. Cliquez sur ce lien pour ouvrir le notebook : TensorFlow Débutant - Colab

    3. Faites une copie dans votre Drive en cliquant sur "Fichier > Enregistrer une copie dans Drive"

    4. Contenu du notebook

    5. Introduction à TensorFlow/Keras
    6. Chargement du dataset MNIST (chiffres manuscrits)
    7. Création d'un modèle de base
    8. Entraînement et évaluation

    9. Visualisation des résultats

    10. Exercices pratiques dans le notebook

    11. Modifier le nombre de neurones
    12. Changer la fonction d'activation
    13. Ajouter des couches supplémentaires
    14. Observer l'impact sur la précision

    Version locale

    1. Installation 

      # Créer un environnement virtuel
python -m venv tf_env

# Activer l'environnement
# Pour Windows
tf_env\Scripts\activate
# Pour macOS/Linux
source tf_env/bin/activate

# Installer les paquets nécessaires
pip install tensorflow numpy matplotlib jupyter

    2. Téléchargement du script

    3. Téléchargez tf_debutant.py

    4. Placez-le dans votre dossier de travail

    5. Exécution du script 

      python tf_debutant.py

    6. Exercices pratiques Le script contient des sections commentées avec "EXERCICE" où vous devrez:

    7. Modifier les hyperparamètres
    8. Ajouter des couches au modèle
    9. Changer les fonctions d'activation
    10. Exécuter le script après chaque modification et observer les résultats

    Partie 2 : Reconnaissance d'images en pratique (1h30)

    Objectifs

    • Utiliser un modèle plus avancé pour la reconnaissance d'images
    • Comprendre le concept de modèle pré-entraîné
    • Créer une interface simple pour tester le modèle

    Version Google Colab

    1. Accès au notebook
    2. Ouvrez ce notebook : Reconnaissance d'images - Colab

    3. Faites une copie dans votre Drive

    4. Contenu du notebook

    5. Chargement d'un modèle pré-entraîné (MobileNetV2)
    6. Prétraitement d'images
    7. Classification d'images test

    8. Création d'une interface avec les widgets Colab

    9. Exercices pratiques

    10. Tester le modèle avec vos propres images
    11. Modifier les paramètres de prétraitement
    12. Comparer différents modèles pré-entraînés
    13. Créer un tableau de résultats comparatifs

    Version locale

    1. Installation des dépendances supplémentaires 

      pip install pillow matplotlib

    2. Téléchargement des fichiers

    3. Téléchargez image_classifier.zip

    4. Décompressez l'archive dans votre dossier de travail

    5. Structure du projet 

      image_classifier/
├── classifier.py         # Script principal
├── images/               # Dossier pour vos images test
│   ├── sample1.jpg
│   ├── sample2.jpg
│   └── ...
└── utils/                # Utilitaires Python
    ├── model_loader.py   # Chargement du modèle
    └── preprocessing.py  # Prétraitement des images

    6. Exécution du script 

      cd image_classifier
python classifier.py

    Le script vous demandera de sélectionner une image dans le dossier images/ et affichera les résultats dans une fenêtre graphique avec matplotlib.

    1. Exercices pratiques
    2. Compléter les fonctions marquées "TODO" dans classifier.py
    3. Ajouter une fonction pour afficher le top 5 des prédictions
    4. Créer une fonction pour comparer plusieurs images côte à côte
    5. Implémenter un système simple de batch processing pour traiter plusieurs images

    Partie 3 : Mini-projet - Classificateur d'images personnalisé (1h)

    Objectifs

    • Adapter un modèle pré-entraîné à vos propres données
    • Comprendre le concept de transfer learning
    • Créer un classificateur personnalisé

    Version Google Colab

    1. Accès au notebook
    2. Ouvrez ce notebook : Classificateur personnalisé - Colab

    3. Faites une copie dans votre Drive

    4. Contenu du notebook

    5. Préparation d'un petit dataset personnalisé (fourni)
    6. Application du transfer learning sur MobileNetV2
    7. Entraînement du modèle adapté

    8. Test et évaluation des performances

    9. Dataset fourni Le notebook télécharge automatiquement un petit dataset contenant 3 catégories:

    10. Téléphones portables
    11. Ordinateurs portables

    12. Tablettes

    13. Exercices pratiques

    14. Adapter le code pour utiliser votre propre petit dataset (10-15 images par classe)
    15. Modifier les paramètres d'entraînement
    16. Expérimenter avec différents modèles de base
    17. Créer une matrice de confusion pour analyser les erreurs

    Version locale

    1. Téléchargement des ressources
    2. Téléchargez mini_projet_classificateur.zip

    3. Décompressez l'archive dans votre dossier de travail

    4. Structure des fichiers 

      mini_projet_classificateur/
├── custom_classifier.py      # Script principal
├── data_loader.py            # Chargement des données
├── visualization.py          # Visualisations
├── dataset/                  # Dataset à télécharger
│   ├── telephones/           # 15 images
│   ├── ordinateurs/          # 15 images
│   └── tablettes/            # 15 images
└── README.md                 # Instructions détaillées

    5. Exécution du projet 

      cd mini_projet_classificateur
python custom_classifier.py

    Le script vous guidera à travers les étapes de transfer learning avec une interface en ligne de commande.

    1. Exercices pratiques
    2. Compléter les parties manquantes dans custom_classifier.py
    3. Modifier les fonctions de data augmentation pour améliorer la généralisation
    4. Implémenter une visualisation avec matplotlib des résultats d'entraînement
    5. Ajouter un mode de test pour évaluer le modèle sur de nouvelles images

    8. Analyse et amélioration

    9. Pertinence des modifications apportées
    10. Analyse des résultats
    11. Propositions d'amélioration
